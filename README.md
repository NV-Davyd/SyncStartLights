# SyncStartLights

Public KSL updater repository for SyncStartLights.

This repository intentionally does not contain mod source code. KSL uses the
GitHub releases from this repository to check versions and download the
packaged `.ksm` release archive.

Current public version: `v1.0.3`.

## Install/update

Install through KSL after the mod is linked in the KSL mod registry. Releases
use tags in KSL's required `v*.*.*` format.
